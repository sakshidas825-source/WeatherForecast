# 🌦️ Weather Forecast Application

A Java-based Weather Forecast Application that retrieves and presents weather information based on the user's location. The application utilizes API integration, location detection, and Object-Oriented Programming principles to provide weather updates in a simple console-based interface.

Developed as part of the **Pinnacle Labs Java Development Internship**.

---

## 📌 Project Overview

This application automatically detects the user's location and displays weather-related information, including temperature, humidity, wind speed, and weather conditions.

The project demonstrates Java networking, API integration, data processing, and OOP concepts.

---

## ✨ Features

- 🌍 Automatic user location detection
- 🌦️ Weather information retrieval
- 🌡️ Temperature display
- 💧 Humidity monitoring
- 🌬️ Wind speed tracking
- ☁️ Weather condition reporting
- 🖥️ User-friendly console interface
- 🏗️ Object-Oriented Programming (OOP) architecture

## ⚙️ How It Works

1. Detects the user's current city using a location service.
2. Connects to a weather API.
3. Retrieves weather information.
4. Processes the received data.
5. Displays weather details in a structured format.
